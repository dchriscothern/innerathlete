# InnerAthlete Context

This file captures the current product direction for the repo.

## Product Positioning

InnerAthlete is a performance intelligence platform organized around:
- biomarkers
- genetics
- S2-style cognition testing
- wellness and readiness monitoring
- privacy-safe individualized recommendations

## Current MVP Shape

The current MVP experience is based on:
- one-page performance summary layouts
- two-page individualized development plan layouts
- big-number biomarker dashboards
- action-plan recommendations by domain

## Primary Domains

### Biomarkers

Core markers currently represented:
- hs-CRP
- Vitamin D
- Ferritin
- Cortisol
- GGT
- Glucose
- Triglycerides
- TSH
- Testosterone

### Genetics

Reporting is grouped by:
- Nutrition & Vitamins
- Exercise & Response
- Sleep
- Recovery & Resilience
- Cognitive Skills

### S2 Cognition

Current cognition model uses:
- Timing Control
- Distraction Control
- Impulse Control
- Improvisation
- Instinctive Learning
- Decision Complexity
- Motor Learning

Banding:
- Basic
- Lower Average
- Average
- Higher Average
- Elite

## Privacy Requirements

This project should remain anonymized.

Never place real athlete-identifying information in the repo, including:
- names
- DOB
- phone or email
- team identifiers that can re-identify a person
- raw medical report identifiers

Use anonymized IDs like `ATH-001`.

## Code Reality

The codebase still contains legacy WAIMS naming in places. The current business and documentation framing should be treated as InnerAthlete while cleanup continues incrementally.
