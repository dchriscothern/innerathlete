## InnerAthlete Brand Assets

This folder is the canonical home for app-safe InnerAthlete branding assets.

Included variants:
- `innerathlete-icon-dark.svg`
- `innerathlete-icon-light.svg`
- `innerathlete-horizontal-dark.svg`
- `innerathlete-horizontal-light.svg`
- `innerathlete-stacked-dark.svg`
- `innerathlete-stacked-light.svg`

Notes:
- These are vector-safe app assets created for the product shell and demo views.
- Use the dark variants on light backgrounds and the light variants on dark backgrounds.
- Do not place athlete-specific report graphics or source-material screenshots in this folder.
